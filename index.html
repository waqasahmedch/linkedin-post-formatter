How does this work? (click to expand)

Neither LinkedIn nor X (Twitter) support native rich text formatting when composing posts. This tool uses Unicode Mathematical Alphanumeric Symbols — characters that visually look bold or italic — which render as formatted text on both platforms. Underline and strikethrough use invisible combining characters appended after each letter. Since it's all real text (not images), it stays formatted when you paste.

@ Mention inserts an @ at your cursor — just type the handle right after. Mentions work on both LinkedIn and X.

Line Spacer inserts an invisible character so LinkedIn doesn't collapse your blank lines.

Platform toggle (in / 𝕏): Switch between LinkedIn (3,000 character limit) and X (280 character limit) — the counter updates automatically.

Tip: Select text first, then click a formatting button. Or use keyboard shortcuts Ctrl/Cmd+B, Ctrl/Cmd+I, Ctrl/Cmd+U.
