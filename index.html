How does this work? (click to expand)

LinkedIn, X (Twitter), Instagram, Threads, and Truth Social all share the same limitation — none of them support native bold or italic formatting when composing posts. This tool uses Unicode Mathematical Alphanumeric Symbols — characters that visually look bold or italic — which render as formatted text across all these platforms. Underline and strikethrough use invisible combining characters appended after each letter. Since it's all real text (not images), it stays formatted when you paste.

@ Mention inserts an @ at your cursor — just type the handle right after. Mentions work on both LinkedIn and X.

Line Spacer inserts an invisible character so LinkedIn doesn't collapse your blank lines.

Platform toggle: Switch between LinkedIn (3,000), Instagram (2,200), Threads (500), Truth Social (500), and X (280) — the character counter updates automatically for each platform.

Tip: Select text first, then click a formatting button. Or use keyboard shortcuts Ctrl/Cmd+B, Ctrl/Cmd+I, Ctrl/Cmd+U.

Your privacy is guaranteed. Everything you type stays entirely in your browser — we never collect, store, or transmit your post content. There is no server, no database, and no tracking. For full transparency, the source code is publicly available on GitHub — you can inspect every line yourself.
